# NS ENTERPRISES — Luxury Furniture Website

A premium furniture **showcase & lead-generation** website for **NS ENTERPRISES**
— _All Types of Luxury Furnitures_, manufactured and delivered **Pan India**.

> This is **not** an ecommerce site. No cart, no checkout, no pricing — only
> beautiful furniture photography, premium branding and enquiry generation.

## ✨ Highlights

- Full-screen hero with floating glass cards + animated statistics
- Premium collections grid, masonry gallery with glassmorphism hover overlays
- "Why Choose Us", About, 4-step Process, auto-sliding testimonials carousel
- Enquiry form that opens a pre-filled WhatsApp message
- Sticky WhatsApp / Call / Enquiry buttons + scroll-to-top
- Framer Motion scroll reveals, counters, hover lift & image zoom
- Mobile-first, fully responsive, SEO meta + JSON-LD structured data

## 🛠 Tech Stack

React 18 · Vite · Tailwind CSS · Framer Motion · Swiper · React Router DOM ·
React Icons · Lucide React. **Tailwind utility classes only** (no Bootstrap/MUI/SCSS).

## 🚀 Getting Started

```bash
npm install
npm run dev      # start dev server (http://localhost:5173)
npm run build    # production build → dist/
npm run preview  # preview the production build
```
